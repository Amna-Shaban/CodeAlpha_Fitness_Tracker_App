

# CodeAlpha - Fitness Tracker App 🏋️‍♂️

A responsive and functional web-based fitness application that allows users to log daily exercises and instantly view calculated metrics on a clean tracking dashboard. This project satisfies **Task 3** for the App Development Internship at **CodeAlpha**.

---

## 🚀 Features
* **Manual Activity Logging:** Users can input specific exercise types, workout lengths, and targeted calorie expenditure metrics.
* **Real-time Dashboard:** Instantly calculates and updates total aggregate workouts and net calorie progress.
* **Persistent Local Storage:** Integrated with browser `localStorage` to ensure fitness tracking history remains intact upon page reload or session loss.
* **Clean & Minimal UI:** Designed with an interface intended for streamlined, clear, and convenient readability.

---

## 🛠️ Tech Stack
* **HTML5:** Structuring layout grids and dashboard containers.
* **CSS3:** Custom interface styling and presentation adjustments.
* **JavaScript (ES6):** State array tracking, local application cache integration, and active DOM modifications.
